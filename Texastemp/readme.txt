"texastemp_forecasting_mFPCA.R": code for mFPC forecasting problem in Texas temperature data analysis.

"texastemp_forecasting_tFPCA.R": code for tFPC forecasting problem in Texas temperature data analysis.

"texastemp_insamplePrediction_mFPCA.R": code for mFPC in-sample prediction problem in Texas temperature data analysis.

"texastemp_insamplePrediction_tFPCA.R": code for tFPC in-sample prediction problem in Texas temperature data analysis.

"texastemp.R": code for full tFPC model with principal components, mean-estimation, etc.
