"EMalg.cpp": the EM algorithm for mFPC model.

"CV_mFPCA_realdata.R": cross-validation algorithm of the mFPC model in Texas temperature data analysis.

"CV_mFPCA.R": cross-validation algorithm of the mFPC model in simulation studies.

"EMinitial_mFPCA_realdata.R": to obtain the initial values of EM algorithm for mFPC in Texas temperature data analysis.

"EMinitial_mFPCA.R": to obtain the initial values of EM algorithm for mFPC in simulation studies.

"mFPCA_Penaltyparameters.R": simplex method for tuning parameters.

"mFPCA_simplex_realdata.R": simplex method for tuning parameters in Texas temperature data analysis.

"Multi_mFPCA.R": multiple utility functions for mFPC model.
