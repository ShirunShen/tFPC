EMalgorithm.cpp: the major code for implementation of tFPC algorithm including Kalman filter and smoother, gradient descent on manifold and expectation maximization algorithm.

Multi_tFPCA.R: the multiple R functions for utilities in both simulation studies and real data analysis, including the construction of Bernstein polynomial basis on triangulations.

CV_tFPCA.R and tFPCA_Penaltyparameters.R: the functions for implementations of cross validation for tuning penalty parameters.

CV_tFPCA_realdata.R and tFPCA_simplex_realdata.R: the functions for implementations of cross validation for tuning penalty parameters of real data analysis.

AIC.R: implementation of AIC approach for tuning the order p of AR model.
